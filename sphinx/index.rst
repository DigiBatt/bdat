Welcome to bdat's documentation!
================================

.. image:: images/bdat_structure.svg

The Battery Data Analysis Toolkit (bdat) provides a set of functions that enable the import and analysis of electrical battery data.
It is designed to interact with a `Kadi4Mat <https://kadi.iam.kit.edu/>`__ instance, but can also be used as a standalone tool.
